Проект: Анализ продаж пиццерии 2015

ОПИСАНИЕ ПРОЕКТА

Полный анализ годовых продаж пиццерии, включающий обработку данных в Python и создание интерактивного дашборда в Yandex DataLens. Проект выполнен как учебное задание, демонстрирующее полный цикл работы с данными.

ЧТО СДЕЛАНО В ПРОЕКТЕ

1. АНАЛИЗ ДАННЫХ В PYTHON
   • Загружены и объединены в единую таблицу данные из 4 исходных CSV-файлов
   • Проведена очистка данных: обработка пропущенных значений, поиск и удаление дубликатов
   • Проанализированы выбросы с помощью z-score и визуальных методов (boxplot)
   • Выполнен статистический анализ: описательная статистика, частотный анализ, корреляции
   • Построено 5+ различных визуализаций (гистограммы, линейные графики, диаграммы рассеяния, boxplot, heatmap)
   • Проведены группировки и агрегации данных с использованием groupby и pivot_table

2. ДАШБОРД В YANDEX DATALENS
   • Создано подключение и загружен объединенный датасет
   • Разработаны 3 интерактивных чарта разных типов:
       1. Столбчатая диаграмма "Выручка по категориям пицц"
       2. Линейный график "Динамика заказов по месяцам"
       3. Сводная таблица "Продажи по видам и размерам пицц"
   • Создан единый дашборд с компоновкой всех визуализаций
   • Настроены фильтры (селекторы) по дате и категориям пицц
   • Реализована интерактивность: переходы по клику между графиками

3. ПУБЛИКАЦИЯ НА GITHUB
   • Создан публичный репозиторий с полной структурой проекта
   • Добавлены все необходимые файлы: Jupyter Notebook (.ipynb), CSV-датасет, README
   • Выполнено два коммита по требованиям задания:
       - Первый коммит: загрузка исходных данных и структуры проекта
       - Второй коммит: финальный анализ и визуализации

ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ

• Python 3.9+
• Библиотеки: pandas, numpy, matplotlib, seaborn
• Yandex DataLens (для создания интерактивного дашборда)
• GitHub (для хранения и версионирования кода)
