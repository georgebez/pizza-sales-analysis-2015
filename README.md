ПРОЕКТ: Анализ продаж пиццерии 2015
ОПИСАНИЕ ПРОЕКТА: Полный анализ годовых продаж пиццерии с визуализацией в Yandex DataLens. Проект выполнен в рамках учебного задания по анализу данных.

Что сделано в проекте
1. Анализ данных в Python
   
-Загружен и объединен датасет из 4 таблиц
-Проведена очистка данных: обработка пропусков, дубликатов, выбросов
-Выполнен статистический анализ и корреляционные исследования
-Построено 5+ различных визуализаций (гистограммы, линейные графики, диаграммы рассеяния, boxplot, heatmap)
-Проведены группировки и агрегации данных

2. Дашборд в Yandex DataLens
Созданы 3 интерактивных чарта:

-Столбчатая диаграмма "Выручка по категориям пицц"
-Линейный график "Динамика заказов по месяцам"
-Сводная таблица "Продажи по видам и размерам пицц"
-Настроены фильтры по дате и категориям
-Реализована интерактивность между графиками
-Создан единый дашборд с управляемыми элементами

3. Публикация на GitHub
Создан публичный репозиторий с полной структурой проекта

-Добавлен Jupyter Notebook с кодом анализа
-Включен датасет в формате CSV
-Написана документация в README файле
-Выполнено два коммита: начальная структура и финальный анализ

Используемые технологии
-Python 3.9+
-Библиотеки: pandas, numpy, matplotlib, seaborn
-Yandex DataLens для визуализации
-GitHub для хранения кода
